
# FastAPI Café Ordering API with Google ADK Integration

## Overview

This project provides a simple backend API for managing a café's menu and orders, built using FastAPI. It also includes instructions for integrating this API with a conversational agent using the Google Agent Development Kit (ADK) Web interface, allowing users to interact with the café service via natural language.

The FastAPI application handles:
* Listing menu items (including IDs, names, prices, and descriptions).
* Placing new orders by specifying dish IDs.
* Retrieving lists of all past orders.
* Fetching details of a specific order by its ID.

The Google ADK agent, configured via the ADK Web UI, uses the API's automatically generated OpenAPI specification to understand and call the available API endpoints.


## Features

* **FastAPI Backend:**
    * `/menu` (GET): Retrieve the full list of available dishes.
    * `/orders` (POST): Place a new order (requires a list of dish IDs).
    * `/orders` (GET): Retrieve all previously placed orders.
    * `/orders/{order_id}` (GET): Retrieve details for a specific order.
    * Automatic interactive API documentation at `/docs`.
    * Automatic OpenAPI specification at `/openapi.json`.
* **Google ADK Agent (via Web UI):**
    * Tools are automatically generated by pointing the ADK Web UI to the API's `/openapi.json` endpoint.
    * Capable of understanding requests like "Show me the menu", "Order item X and item Y", "List my orders".

## Setup and Running Instructions

### Prerequisites

* Python 3.13 or higher
* Google ADK installed (`pip install -U google-adk`)
* FastAPI dependencies (`pip install fastapi uvicorn`)

### Step 1: Start the FastAPI Backend

First, run the FastAPI café API server:

```bash
# Navigate to the project directory
cd /path/to/google-adk-apps/examples/adk_agent_order_cafe

# Start the FastAPI server
python api_cafe.py
```

This will start the server on http://127.0.0.1:5000. You can verify it's working by accessing:
- API documentation: http://127.0.0.1:5000/docs
- OpenAPI specification: http://127.0.0.1:5000/openapi.json

### Step 2: Launch the Google ADK Web Interface

In a new terminal window, run:

```bash
# Start the ADK web interface
adk web
```

This will launch the Google ADK Web UI in your default browser. If it doesn't open automatically, navigate to http://localhost:8000 or http://127.0.0.1:8000 in your browser.
